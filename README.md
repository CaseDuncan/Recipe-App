# RECIPE APP

## Author: Oluoch Duncan 24th June 2022

## Project Description

The project is about a simple recipe app that makes using of JavaScript fetch API to retrieve recipes for meals  from MealDB API. 

Retrieving:
1. Name of the Meal
2. Ingredients 
3. Preparation Procedure
